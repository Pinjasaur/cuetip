# cuetip

Simple CSS Tooltips

# Getting Started

## Browser Support

cuetip relies on CSS3 `calc()` which is [IE9+ (with no Opera Mini support)][ciu-calc].

## Installation

Bower, NPM, or drag 'n' drop.

## Configuration

Tons of Sass variables to configure it to how you'd like.
